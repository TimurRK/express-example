## Express Example

## Dependencies

* [MongoDB](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-ubuntu/)
* [NodeJS 12.14.1](https://www.ubuntuupdates.org/ppa/nodejs_12.x?dist=bionic)

## Installation

```bash
npm i
```

## Running

### development

```bash
npm run start:dev
```

### beta

```bash
npm run build
npm run start:beta
```

### production

```bash
npm run build
npm run start:prod
```
